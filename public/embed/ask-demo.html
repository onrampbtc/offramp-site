Ask Offramp — embed demo

This page demonstrates both widget modes. On localhost the iframe loads from this origin (the dev server); in production it loads offrampgold.com/ask?embed=1.

1. Launcher mode

A floating button sits bottom-right (look down-right now). One line anywhere in the page: 

<script src="https://offrampgold.com/embed/ask.js"
        data-cta="What's your gold worth?"
        data-brand="goldbuyer"
        data-mode="launcher"
        data-accent="#B45309" async></script>

2. Inline mode

The button renders exactly where the script tag sits:

Both open the same modal (dark scrim, ESC or scrim-click closes, focus stays in the dialog). The close control inside the flow posts {type:'ask-offramp:close'} back to this page.
